## 개요

CI/CD 를 하기 위한 아마존 서비스이다. 

단독으로 사용할 수 없으며 [AWS CodeCommit](AWS%20CodeCommit), [AWS CodeBuild](AWS%20CodeBuild), [AWS ECR](AWS%20ECR), [AWS Deploy](AWS%20Deploy) 을 연동하여 지속가능한 통합과 배포를 하기 위하여 사용한다.
